---
## Description

_HotHotHot_ est une application dont le rôle est d’être une interface d’information et de gestion d’un système domotique.

Cette application reçoit des données en provenance de différents capteurs comme par exemple :
- température
- humidité
- état d’un périphérique

Elle affiche ces données, leur historique et propose des suggestions. De plus, elle émet des alertes à l’attention des utilisateurs.
